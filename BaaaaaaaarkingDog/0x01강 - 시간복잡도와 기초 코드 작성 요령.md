# 0x01강 - 시간복잡도와 기초 코드 작성 요령



## 0x00 시간, 공간복잡도

| N의 크기      | 허용 시간복잡도 |
| ------------- | --------------- |
| N <= 11       | O(N!)           |
| N <= 20       | O(2^N)          |
| N <= 100      | O(N^4)          |
| N <= 500      | O(N^3)          |
| N <= 3000     | O(N^2 * lgN)    |
| N <= 5000     | O(N^2)          |
| N <= 1000000  | O(NlgN)         |
| N <= 10000000 | O(N)            |
| 그 이상       | O(lgN), O(1)    |
